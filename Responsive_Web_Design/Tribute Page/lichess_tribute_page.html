Tribute to Thibault Duplessis, Lichess.org Founder

Lichess.org

Lichess.org has grown to be one of the most popular online chess platforms in the world. Offering free and open-source resources for players ranging from Chess Masters, to Chess Beginners.


Thibault Duplessis, Founder of Lichess.org

Bio

Thibault Duplessis: is a French programmer who created Lichess.org in his free time. From writing the source code to running the servers, he built the foundation for the platform we know today. He has since received help from colleagues, but his dedication played a key role in Lichess.org's success.

Github Repositories

Lichess.org is committed to open-source development. All of the source code can be found on Github. For those interested in contributing or learning more about the development process, check out the following links:
